## README

# European Financial Data Space Status

Research documentation on the European Financial Data Space (EFDS), covering ESAP, ESEF, and FIDA regulatory frameworks.

## Overview

This repository contains a comprehensive status report on the European Financial Data Space initiative, including:

- **ESAP** (European Single Access Point) - Centralized EU financial data portal
- **ESEF** (European Single Electronic Format) - iXBRL reporting standard
- **FIDA** (Financial Data Access) - Open finance framework proposal
- **ESRS** (European Sustainability Reporting Standards) integration
