## Metadata Structure

Each publication entry contains:

- Basic bibliographic information (title, DOI, year, type)
- Abstract (where available)
- Citation metrics and trends
- Open access status and PDF links
- Author information and institutional affiliations
- Subject classifications (topics, concepts, keywords)
- Publication venue details
- Funding information (where available)

## Usage

The JSON file can be processed using Python, R, or any JSON-compatible tool for:

- Bibliometric analysis
- Citation network visualization
- Collaboration analysis
- Research trend identification
- Impact assessment

All data retrieved from [OpenAlex](https://openalex.org/), an open catalog of scholarly papers, authors, institutions, and venues.
