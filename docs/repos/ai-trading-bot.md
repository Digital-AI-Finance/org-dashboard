## README

# AI Trading Bot

A sophisticated machine learning-powered algorithmic trading system designed for real-time market analysis and automated trading decisions.

## Features

- Real-time market data processing
- Multiple ML models (LSTM, Transformer, Random Forest)
- Risk management and portfolio optimization
- Backtesting framework
- Live trading integration with major exchanges

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```python
from ai_trading_bot import TradingBot

bot = TradingBot()
bot.run()
```
