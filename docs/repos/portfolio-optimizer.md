## README

# Portfolio Optimizer

An advanced AI-driven portfolio optimization engine that combines Modern Portfolio Theory with cutting-edge machine learning techniques.

## Features

- Mean-variance optimization
- Black-Litterman model
- Risk parity strategies
- ML-enhanced return predictions
- Multi-objective optimization
- ESG-aware portfolio construction

## Installation

```bash
pip install portfolio-optimizer
```

## Example

```python
from portfolio_optimizer import Optimizer

optimizer = Optimizer()
weights = optimizer.optimize(returns, risk_free_rate=0.02)
```
